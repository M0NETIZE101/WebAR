# WebAR Dinosaur - QR Tracking with Three.js

## 🦕 Overview
A WebAR application that displays an animated 3D dinosaur when scanning a QR code marker using Three.js and WebXR.

## ✨ Features
- WebXR AR tracking with Three.js
- QR code detection for model placement
- Animated dinosaur with GLB format support
- Responsive and works on iOS/Android
- Extensible for multiple models

## 📋 Prerequisites
- HTTPS server (local or production)
- Modern browser with WebXR support
- Camera access permission
